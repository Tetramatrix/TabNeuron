Privacy Policy – Tetramatrix Connector

⬇️ ⬆️

Data Collection

Tetramatrix Connector does not collect, store, or transmit any personal user data to external servers or third parties.

Local Data Processing

The extension reads tab URLs, titles, and page content exclusively to forward them to the Tetramatrix desktop application (TabNeuron) running locally on the user's own machine via localhost. This data never leaves the user's device.

Permissions

Permissions (tabs, tabGroups, scripting, storage, activeTab) are used solely to enable communication between Chrome and local Tetramatrix applications. No data is logged or stored remotely.
